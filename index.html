Hydro-Topo Features Interactive Maps

These interactive maps showcase the hydro-topographic features extracted using the Hydro-Topo Features Python package.

The maps allow exploration of HAND (Height Above Nearest Drainage), Slope, and EDTW (Euclidean Distance To Water) across different geographic regions.

Danube River Basin Interactive Map

The Danube River Basin map showcases the hydro-topographic features across one of Europe's most important river systems, spanning multiple countries and diverse terrain.
